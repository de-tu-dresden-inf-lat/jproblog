### v0.1.0
*(2016-09-01)*
* requires Java 8
* downloads ProbLog 2.1
* if Python 2.7 or 3.2 is not installed, it downloads Jython 2.7.0 from the Central Repository, and it uses Jython instead
* build commands:
```
$ mvn clean install
```
* release: `jproblog/target/jproblog-0.1.0.jar`
